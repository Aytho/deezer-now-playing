Configuring the OBS player

The file player.html cannot be found in the folder player/src/.

To begin :

  1. Go to the folder player/src/ of your extension
  2. Copy the following files :
    • player-example.html → player.html
    • player-example.css → player.css
    • player-example.js → player.js
  3. Reload this page
  4. Customize your player according to your needs !

Tip :

The files player-example.* contain a simple basic example. Once copied into player.*, you can modify them without affecting the original examples.
